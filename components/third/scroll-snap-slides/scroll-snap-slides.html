Ajuste del scroll (Slides)

Lo que haremos en este ejercicio sera controlar ciertos elementos simulando una presentación, definiremos que nuestros elementos se queden en la mitad y podamos pasar de uno a otro con tan solo deslizar un poco la rueda del ratón o el espacio

Para que este ejercicio funcione sin problemas debemos tener varios puntos en cuenta:

  1. Tener un contenedor, aunque podría ser el body, es mejor hacer uno específicamente
  2. Necesitamos dos contenedores, en el primero vamos a definir la altura y anchura, ademas que definimos el overflow en hidden ya que solo queremos que se vea uno
  3. Debemos tener altura y anchura definida por lo menos en nuestro segundo contenedor
  4. En el eje que queremos controlar el efecto debe ser scroll esto lo definiremos en el segundo contenedor

Propiedades

Segundo contenedor

scroll-snap-type Es la propiedad que nos permite controlar el comportamiento de nuestra barra de navegación.
esta propiedad recibe dos valores

none -> No hacer el efecto. Por defecto

Primer valor
  • X o inline - Definimos que queremos controlar el desplazamiento del eje horizontal
  • Y o block -> Para controlar el scroll vertical
  • both -> Para controlar ambos
En el segundo valor definimos que efecto queremos, recibe uno de los siguientes valores
  • mandatory -> Con desplazar un poco el scroll pasara o se devolverá el slide
  • proximity -> De esta manera nuestro movimiento debe ser mayor para que el desplazamiento ocurra, debe estar mas cerca

Slides

A nuestro elemento hijo debemos definir la alineación que este tendrá cuando se haga el efecto

scroll-snap-align es la propiedad que nos lo permite, esta recibe las palabras clave:

  • none Por defecto
  • end -> Se alineara al final
  • start -> Al inicio
  • center -> nuestro elemento estará en el centro

Segundo ejemplo

En este ejemplo haremos el efecto en nuestro segundo eje, en el Y

Recapitulemos los "pasos" para que este efecto funcione

  1. Definir el ancho o alto de nuestro elemento dependiendo el eje que queremos aplicarlo, lo recomendado es hacerlo a ambos
  2. Que la propiedad overflow en el eje que queremos controlar tenga el valor scroll
  3. Definir la propiedad scroll-snap-type en nuestro contenedor
  4. Definir la alineación con la propiedad scroll-snap-align en nuestro elemento hijo

Texto con gradiente

Asi es, a nuestros textos también podemos aplicar un color con las funciones de gradiente, sin embargo esta aun experimental, por lo que si la vamos a usar debemos poner el soporte para los navegadores

Para aplicar este efecto a nuestros textos lo que debemos hacer es definir el gradiente normalmente, en la propiedad background-image y definimos el gradiente que queremos, luego, usando background -clip pondremos el valor de text que debemos definir con los soportes. Por ultimo haremos que nuestro texto ignore la propiedad color, esto lo hacemos con text-fill-color y su valor en transparent, que igualmente debemos definir con soportes
