Modelo de Caja

Es la forma en que CSS ve a los elementos HTML. Como is fueran cajas con las siguientes propiedades:
  1. Content:

    El contenido del elemento HTML, tomando en cuenta sus dimensiones: width ancho height alto

  2. Border:

    Lo que delimita cada uno de los elementos HTML

  3. Padding:

    Son las distancias internas, la distancia del borde al contenido

  4. Margin:

    Las distancias externas, la distancias entre el elemento y sus hermanos o padres

Lados dentro del modelo de caja:
  1. Top -> Arriba
  2. Right -> Derecha
  3. Bottom -> Abajo
  4. left -> Izquierda

Estas cuatro propiedades tienen su shorthand :

        
          inset: 10% 20% 30% 40%;

El orden en que se definen es: Top, Right, Bottom, Left

Border

Color

Para definir el color de nuestro borde

        
          border-color: green;

Este es el shorthand para aplicar a los cuatro lados nuestro color, pero si necesitamos tan solo aplicar el color a uno de las lados lo podemos hacer:

  • border-top-color
  • border-right-color
  • border-bottom-color
  • border-left-color

Ancho de la linea

Podemos usar cualquier unidad de medida, pero en la mayoría de los casos es mejor en pocos pixeles para que no quede tan mal a la vista, ademas, tenemos ya predefinidos ciertos tamaños que podemos definir con border-width

Thin -> Delgado (1px) 
        
          border-width: thin;
Medium -> Mediana (3px) 
        
          border-width: medium;
Thick -> Gruesa (5px) 
        
          border-width: thick;

Este igualmente es un shorthand, si necesitamos un tamaño distinto en alguno de nuestros lados lo podemos definir con:

  • border-top-width
  • border-right-width
  • border-bottom-width
  • border-left-width

Estilo de la linea

Definiremos como queremos que nuestra linea muestre el borde

Mas usados:

        
          Solid
La que he estado usando para presentar los ejemplos. Hará que nuestro borde se muestre con una linea recta y continua 
        
          dotted -> punteada
        
      
        
          dashed -> discontinua
Estas dos opciones son algo parecidas, sin embargo podemos ver un poco la diferencia en lo largo de las lineas internas. Podemos usarlo según nuestros gustos

Tenemos otras Opciones que podemos visitar por si las necesitamos 

        
          Combinado

Podemos combinar nuestros estilos o poner uno en cada lado:

  • border-top-style
  • border-right-style
  • border-bottom-style
  • border-left-style
Debemos tener en cuenta que aunque todos son shorthands no siempre debemos usarlos ya que si solo vamos a definir uno de los lados y lo hacemos con esta propiedad consumiremos recursos innecesariamente. Lo ideal seria que si necesitamos solo uno de los lados lo hagamos con su propiedad especifica
Border

Podemos usar el shorthand para definir todos los estilos de nuestro borde

        
          border: 2px dashed blue;

El orden para definir los estilos son: border-width - border-style - border-color

Si no ponemos el estilo del color, el navegador toma el que tiene el texto en su interior

Margin y Padding

        
                      
            padding: 20px;
            margin: 20px;
Estos también son shorthands, podemos poner padding independientemente en cada lado con:
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
e igualmente a margin:
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

En los shorthands podemos poner mas de un valor, lo que define a que lados de nuestro elemento se les aplicaran el estilo

Los valores aplicara los estilos asi:

  1. valor -> A todos los lados
  2. valores
    • Primer valor -> verticales (top, bottom)
    • Segundo valor -> horizontales (left y right)
  3. valores
    • primer valor -> top
    • segundo valor -> left y right
    • tercer valor -> bottom
  4. valores
    • primer valor -> top
    • segundo valor -> right
    • tercer valor -> bottom
    • cuarto valor -> left
Cuando el valor es 0 no es necesario especificar la medida

Cajas de linea VS Cajas de bloque

Cajas de linea

  • Ocupan el espacio necesario para mostrar su contenido.
  • No tienen dimensiones modificables (width, height)
  • Permiten a otros elementos a su lado
  • Padding y margin solo empujan a elementos adyacentes en horizontal. nunca en vertical (ni top, ni bottom)
  • El padding y margin si se aplicara a nuestro elemento, pero el siguiente no se moverá. Por que nuestro elemento estará sobre este
  • El margin no tiene efecto ni en top ni en bottom

Caja de bloque

  • Ocupan todo el ancho disponible, lo que genera saltos de linea.
  • Tienen dimensiones modificables (alto, ancho)
  • No permiten otros elementos a su lado (aunque especifique un ancho, siguen ocupando todo el espacio disponible, igualmente generando saltos de linea)

Inline

        
          display: inline;

Asi nuestro elemento tomara las características de una caja de linea

block

        
          display: block;

Nuestro elemento tomara las características de una caja de bloque

Display

Determina como se va a comportar nuestro elemento

Opciones:
  • inline
  • block
  • inline-block -> Se comporta como un elemento de linea pero acepta modificar sus dimensiones (height, width)
  • none -> Nuestro elemento no se vera, pero igualmente cargara.
  • table
  • list-item
  • flex
  • grid

Visibility

  • hidden -> ocultara nuestro elemento pero conservara que estaba usando
  • visible -> haremos que nuestro elemento se vuelva a ver

Overflow

Definimos el comportamiento que tendrá nuestro contenido cuando sobresalgo de nuestro elemento

  • Visible -> por defecto, hará que nuestro contenido se vea aunque sea por fuera de nuestro elemento
  • Hidden -> oculta el contenido que sobresalga
  • scroll -> Creara las dos barras de navegación, el tamaño sera automático, es decir hasta que se vea nuestro contenido por completo. Si no es necesaria una de las dos se vera deshabilitada
  • auto -> El navegador decidirá cual es la mejor opción
  • Podemos decidir el comportamiento por ejes:
    • overflow-y: scroll -> vertical
    • overflow-x: hidden -> horizontal
    de esta manera solo se vera la barra de navegación vertical

Box-Sizing -> Tamaño de Caja

Define como el elemento va a calcular el alto y ancho

        box-sizing: content-box;

Valor por defecto, tomara el tamaño de la caja a partir del inicio del contenido sin el padding ni el borde 

        box-sizing: border-box;

El tamaño de la caja tendrá en cuenta el padding y el border

lo que hemos definido en estas propiedades serán considerados dentro de nuestro width y height.
Los errores que se pueden dar usando el valor por defecto es que no se va a tomar como punto fijo el tamaño que hemos definido, ira aumentando según nuestros otros estilos
Si definimos el tamaño de la caja desde el borde nos olvidaremos de hacer cálculos con el padding y borde que queramos agregarle a nuestro contenedor
inicializar

Desde el inicio pondremos nuestros elementos con el tamaño de la caja desde el borde. Ya que lo raro es que necesitemos el de por defecto, si lo necesitamos, podemos definir esta propiedad en nuestro elemento 

        
          html {
            box-sizing: border-box;
          }

          *,
          *::before,
          *::after {
            box-sizing: inherit;
          }

Asi le aplicamos el tamaño de caja a nuestro documento html y hacemos que todos los elementos hereden este tamaño
