Margin y Padding

Padding

Son las distancias internas, la distancia del borde al contenido

Margin

Las distancias externas, la distancias entre el elemento y sus hermanos o padres

      
        padding: 20px;
        margin: 20px;

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
e igualmente margin:
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

En los shorthands podemos poner mas de un valor, lo que define a que lados de nuestro elemento se les aplicaran el estilo

Los valores aplicara los estilos asi:

  1. valor -> A todos los lados
  2. valores
    • Primer valor -> verticales (top, bottom)
    • Segundo valor -> horizontales (left y right)
  3. valores
    • primer valor -> top
    • segundo valor -> left y right
    • tercer valor -> bottom
  4. valores
    • primer valor -> top
    • segundo valor -> right
    • tercer valor -> bottom
    • cuarto valor -> left
Cuando el valor es 0 no es necesario especificar una medida
