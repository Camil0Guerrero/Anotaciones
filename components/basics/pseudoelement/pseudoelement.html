PseudoElementos

¿Que hacen?

Dan estilos a partes especificas de un elemento, usamos el :: para diferenciarlos de las PseudoClases

Content es al propiedad que nos permite agregar el contenido. Lo que queramos agregar debe ir entre comillas
Si asignamos nuestro pseudoElemento con solo : igualmente nuestros estilos se aplicaran, sin embargo, es buena practica hacerlo con :: para que se pueda identificar y diferenciar rápidamente de las PseudoClases

::Before

Nos permite agregar estilos o contenido antes de nuestro elemento

      
        .example::before {
          content: "$";
        }

::After

Luego de nuestros elementos

      
        .example::after {
          content: "COP";
        }

10.000

::first-letter

Aplicaremos los estilos a la primera letra

      
        .example::first-letter {
          font-weight: bold;
          font-size: 32px;
        }

Letra Capital se le llama

::first-line

Para la primera linea

      
        .example::first-line {
          color: coral;
        }

::selection

      
        .example::selection {
          color: greenyellow;
          background-color: black;
        }

Este PseudoElemento nos ayuda a darle estilos a el texto que el usuario seleccione

::placeholder

Nos ayuda a darle estilos a el texto del placeholder

      
        .example::placeholder {
          color: yellow;
          background-color: rosybrown;
        }
