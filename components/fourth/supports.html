@Supports

Feature Queries, son reglas CSS que le preguntan al navegador si soportan o no una determinada propiedad o valor

Por ejemplo una propiedad para los hijos cuando aplicamos grid que no esta soportada es subgrid, en este caso si nuestro navegador soporta esta propiedad aplicara los estilos que hemos definido 

      
        @supports (grid-template-columns: subgrid) {
          // Código CSS
        }

También podemos hacerlo para el caso contrario, cuando el navegador no soporte esta propiedad o característica 

      
        @supports not (grid-template-columns: subgrid) {
          // Código CSS
        }

Si queremos evaluar varias propiedades podemos hacer uso de los operadores and y or 

      
        @supports (display: grid) and (grid-template-columns: subgrid) {
          // Cuando acepte ambas propiedades
        }

        @supports (display: grid) or (grid-template-columns: subgrid) {
          // Debe aceptar una de las dos propiedades
        }
