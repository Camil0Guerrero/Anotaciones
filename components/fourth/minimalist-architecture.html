Arquitectura Minimalista

Esta arquitectura es planteada por @jonmircha

En una sola hoja de estilos define los estilos los cuales los organiza de este modo:

  1. Custom Properties -> Declaración de variables
  2. Reset -> Normalizar y definir valores iniciales a etiquetas
  3. Components -> Elementos que se pueden usar en otro proyecto
  4. Utilities -> Clases declarativas enfocadas solo a una función
  5. Site Styles -> Estilos particulares de cada proyecto
