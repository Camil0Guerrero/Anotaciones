Responsive Grid

Esta grid esta hecha para practicar el concepto del Responsive Design, ademas estamos aplicando una lógica parecida a la que nos brindan frameworks muy conocidos

Se usan 12 o 16 columnas ya que estos números son múltiplos muy divisibles por lo tanto se facilitaba el trabajar con estas columnas

De a poco se dejo de usar las 16 columnas y ya casi todas las libreras usan las 12 columnas casi como estandar

la sumatoria de nuestros elementos que queremos acomodar con esta manera debe sumar 12 ya que si es mas este elemento pasara a la fila de abajo

Para asignar el valor de cuanto va a ocupar nuestro elemento en las columnas lo vamos a definir en porcentajes, esto lo calcularemos con una regla de tres 

      
        12 --> 100%
        ?col --> ???%
Cuando el numero no sea exacto lo recomendado es poner todos los decimales que nuestra calculadora nos brinde

Una cosa que es importante entender es que al crear clases dentro de las media queries, estas se aplicaran cuando la condición se cumpla. Por lo que haciendo uso de ellas podemos hacer que nuestro elemento se vea responsivo
