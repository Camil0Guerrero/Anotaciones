Overflow & Visibility

Overflow

Definimos el comportamiento que tendrá nuestro contenido cuando sobresalgo de nuestro elemento

Opciones

Visible

por defecto, hará que nuestro contenido se vea aunque sea por fuera de nuestro elemento

Hidden

Oculta el contenido que sobresalga

Scroll

Creara las dos barras de navegación, el tamaño sera automático, es decir hasta que se vea nuestro contenido por completo. Si no es necesaria una de las dos se vera deshabilitada

Auto

El navegador decidirá cual es la mejor opción

Ejes

Podemos decidir el comportamiento por ejes:

  • overflow-y: scroll -> vertical
  • overflow-x: hidden -> horizontal

de esta manera solo se vera la barra de navegación vertical

Visibility

Esta propiedad nos ayuda para mostrar u ocultar un elemento sin cambiar su disposición

Opciones

Hidden

Ocultara nuestro elemento pero conservara el espacio que estaba usando

visible

Haremos que nuestro elemento se vuelva a ver
