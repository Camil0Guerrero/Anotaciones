Posicionamiento

Position

Nos permite modificar la disposición natural de cualquier elemento del documento HTML

El navegador coloca cada elemento teniendo en cuenta el orden en el que aparece en el documento y su tipo de visualización, dependiendo si es un elemento en linea o bloque

Al definir cualquier tipo de Posicionamiento se nos activan las propiedades:

  • Top (vertical -- eje Y)

    + baja el documento -> - lo sube

  • Left (horizontal -- eje X)

    + lo mueve hacia la derecha -> - lo mueve hacia la izquierda

  • Bottom (vertical -- eje Y)

    + mueve hacia arriba el elemento -> - lo baja

  • Right (horizontal -- eje X)

    + mueve hacia la izquierda -> - lo mueve hacia la derecha

  • Z-index (profundidad -- eje Z)

Estas no están disponibles es nuestro posicionamiento por defecto (static)

Las propiedades top y left tienen preferencia sobre bottom y right

Tipos de Posicionamiento

Static

Es el valor por defecto, significa que el elemento respeta el flujo del documento, por lo que no se puede mover ni posicionar 

      
        .static {
          background-color: yellow;
          position: static;
          width: 200px;
          top: 10px;
          left: 10px;
        }
Como vemos, ni el top ni el left tienen efecto

Relative

El elemento conserva sus dimensiones y posición original en el flujo del documento. Si lo movemos toma como referencia su posición original

Aunque movamos nuestro elemento, en donde este estaba se quedara el vació, como si siguiera allí

      
        .relative {
          background-color: turquoise;
          bottom: 50px;
          left: 100px;
          position: relative;
          right: 50px;
          top: 10px;
          width: 200px;
        }
bottom y right no tienen efecto ya que hemos definido top y left
Podemos ver como nuestra caja se fue 10px hacia abajo y 100px hacia la derecha de donde se encontraba

Absolute

El elemento pierde sus dimensiones y posición original en el flujo del documento. Si se mueve puede tomar como referencia 2 elementos:

  1. El primer ancestro con posicionamiento relativo
  2. Si no encuentra un ancestro relativo, se mueve respecto del documento HTML
Al perder sus dimensiones se las debemos dar explícitamente
Sin Ancestros relativos 
      
        .absolute {
          background-color: tomato;
          position: absolute;
          width: 20%;
          height: 30px;
          /* right: 0;
          bottom: 0; */
        }
Si habilitamos nuestras 2 propiedades de posicionamiento, nuestro elemento se movera hacia la parte inferior derecha pero del inicio de nuestro documento

Con ancestro relativo

        
          .absolute-child {
            background-color: lightcoral;
            position: absolute;
            right: 0;
            bottom: 0;
          }
De esta forma nuestro elemento se esta ubicando según su ancestro, en este caso lo hemos puesto en la parte inferior derecha

Fixed

El elemento pierde sus dimensiones y posición original en el flujo del documento. Si lo movemos este toma como referencia el documento HTML y queda fijo en la posición, cuando el scroll se mueve el elemento no lo hará, queda fijo 

        
          .fixed {
            background-color: lightgreen;
            /* position: fixed; */
            width: 200px;
            height: 30px;
            top: 30px;
            left: 80%;
          }
Con top y left definimos donde se quedara nuestro documento, este durante todo el documento estará allí

Sticky

Es una combinación de posicionamiento relative y fixed, para este posicionamiento las propiedades top, right, bottom y right no mueven el elemento, sirven como un punto de referencia. Mientras no lleguen a ese valor el elemento se comporta como relative, cuando llega se convierte en fixed

Para que este comportamiento funcione el elemento sticky debe ser hijo directo del body o sue elemento contenedor debe tener dimensiones definidas y solo sera sticky dentro de las dimensiones de su contenedor padre
Caja Pegajosa
        
          .sticky {
            background-color: lightskyblue;
            /* position: sticky; */
            top: 20px;
          }

Cuando llegue a 20px del top, de arriba, es cuando empieza a tener el comportamiento de un elemento con posicionamiento fixed
