Selectores

Básicos

  1. Etiquetas -> Elementos HTML
  2. Identificadores -> Atributo id: #
  3. Clases -> Atributo class: .

Ejemplos:

        
          p {
            color: midnightblue;
          }

Si lo hacemos con la etiqueta, el problema es que TODAS las etiquetas de nuestro documento tendrán el estilo 

        
          #paragraph {
            background-color: skyblue;
          }

Con el id es posible, pero se considera un antipatrón, una mala practica ya que solo una etiqueta debería tener el id. ¿ y si queremos aplicar el mismo estilo a otros elementos ?

Los ids tienen mayor peso (especificidad) que los demás selectores por lo que podríamos presentar problemas al usarlos para aplicar estilos

CSS

        
          .bg-color {
            background-color: rgb(29, 53, 62);
          }

          .text-20 {
            font-size: 20px;
          }

          .text-white {
            color: white;
          }

HTML

        
          <p class="bg-color text-20 text-white"> Selección con clase </p>

Resultado:

Selección con clase

Las clases son La mejor opción para dar estilos ya que no solo debemos ponerlas en una etiqueta lo que nos permite reutilizar nuestro código CSS. Al igual que un etiqueta puede tener varias clases

Tratar que los nombres de las clases sean alusivas a los estilos que aplicaremos con ellas

Avanzados

A los elementos que usamos como referencia no se le aplican los estilos

Hijos directos >

        
          .hijos-directos > li

Los hijos directos son aquellos que se encuentran justo después de nuestro elemento padre. Si el elemento si es un hijo pero no esta definido debajo de nuestro elemento padre, El estilo no se aplicara.

Hijos descendientes

        
          .hijos-descendientes b

Los hijos descendientes son los todos los elementos que están dentro de nuestro elemento padre no importa el lugar donde se encuentra el elemento

Hermanos siguientes ~

        
          .hermanos-general ~ li

Los hermanos son los elementos que se encuentran en el mismo nivel

Hermanos adyacentes +

        
          .hermanos-adyacentes + li

Los estilos se le aplicaran solo al siguiente elemento con la etiqueta li

Por referencia

Accederemos a nuestros elementos por medio de sus atributos

General []

        
          .selectores-atributos a[href]

Accederemos a todos los elementos a que tengan el atributo href

Atributo con valor

        
          .selectores-atributos a[target="_blank"]

Podemos ser mas específicos poniendo el valor que queremos que el atributo tenga

Buscando una palabra *=

        
          .selectores-atributos a[href*="palabra"]

Combinando estos símbolos podemos aplicar estilos cuando un elemento tenga en su atributo la palabra que buscamos

Valor de inicio ^=

        
          .selectores-atributos a[href^="http:"]

Tomamos todos los elementos que su atributo inicie con el valor que definimos

Valor final $=

        
          .selectores-atributos a[href$=".com"]

Los estilos se aplicaran a elementos que su atributo finalicen con nuestro valor(.com)

Buscar entre espacios ~=

        
          .selectores-atributos a[class~="cursos"]

Aplica el estilo si contiene el texto en cuestión en una lista separada por espacios

Buscar entre espacios con guion |=

        
          .selectores-atributos a[class|="menu"]

Buscara nuestro texto en una lista separada por espacios y que luego de nuestro texto venga un -; por ejemplo: menu-item

Selector universal *

        
          *
          
          .selector-universal *

La propiedad se aplicara para todos los elementos que estén dentro de nuestro punto de referencia, si no lo hay, se aplicaran a todo el documento

Agrupar selectores ,

      
        .form-agrupar-selectores input[type="text"],
        .form-agrupar-selectores input[type="email"],
        .form-agrupar-selectores textarea{}

Si queremos usar los mismos estilos para varios elementos debemos separar los selectores con una (,)
