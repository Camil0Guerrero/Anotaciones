Float & Clear

Float

Al hacer un elemento flotante haremos que este se salga de la disposición natural, (que vaya uno tras otro). Es como si empezara a levitar.

Si no hemos especificado el ancho y alto de nuestro elemento al hacerlo flotante se desaparecerá el color de fondo

Usar esta propiedad Ya es mala practica. Hay módulos para maquetar, queda mucho mejor y no tenemos que romper con la disposición de nuestros elementos

Esta propiedad tiene dos opciones, right y left las cuales definirán donde nuestro elemento se acomodara 

      
        .example {
          float: right;
        }

Clear

Hará que el siguiente elemento de cuando usamos los float, se baje y siga con el flujo normal, da como un salto de linea a los elementos que no usan el float

Podemos usarlo para que posicione correctamente hacia right, left y both (ambos)
