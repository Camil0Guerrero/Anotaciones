Custom properties

Al igual que en los lenguajes de programación podemos definir variables, básicamente es un espacio reservado en nuestra memoria que va a almacenar un dato

Estas variables también tienen un alcance, un ámbito donde existirá, este sera el selector en donde se declare y todos sus elementos subsecuentes (toda la cadena de hijos en la estructura HTML que tengamos)

Ámbito -> Espacio de existencia

Todas las variables se deben declarar en un selector, Igualmente se puede hacer en el elemento HTML usando el atributo style, igualmente tendrá el mismo uso

Para declarar una variable debemos hacerlo con -- y luego de los : el valor que guardara --name-variable: value;
Para hacer uso de nuestra variable debemos hacer uso de la función var, entre parentesis ponemos el nombre de la que necesitamos. Igualmente con --
Si definimos dos veces la misma variable, entra en juego el algoritmo de CSS, la cascada y la especificad.
Podemos pasar un segundo parámetro por si nuestra variable no se puede utilizar o hay errores. color: var(--primary-color, orange); 
        
          --primary-color: gray;
          --font-size: 32px
          background-color: var(--primary-color);
        
      
          
            background-color: var(--default-bg-color);
            
            /* Variable declarada con el atributo style */
            border: thick solid var(--border-color);
          
        
            
              --text-color: navy;
              color: var(--text-color, orange);
              font-size: var(--font-size);
              border: thin solid var(--border-color);

Para todo el documento

Para que las variables estén disponibles para todo nuestro documento, seria declararlas en el html, pero en CSS el nodo principal es el root, hace referencia al html solo que con mas peso, tiene mas especificad

Aunque lo podríamos hacer en nuestra etiqueta principal que es el html, esta tendrá menos especificad por ser una etiqueta, en cambio nuestro root es una PseudoClase, por esto es que lo ideal es usarlo 

        
          :root {
            --default-bg-color: lightsalmon;
          }
