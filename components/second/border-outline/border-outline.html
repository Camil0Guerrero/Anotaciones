Border y Outline

Bordes Redondeados

Nos ayuda para que las esquinas queden con un estilo de redondeo

Usaremos el Shorthand de 4 propiedades

  • border-top-left-radius -> Arriba izquierda
  • border-top-right-radius -> Arriba derecha
  • border-bottom-left-radius -> Abajo izquierda
  • border-bottom-right-radius -> Abajo derecha
border-radius: 2rem;
border-radius: 2rem 4rem;
border-radius: 2rem 4rem 0;
border-radius: 4rem 3rem 2rem 1rem;

Bordes elípticos

Cuando estamos dando el redondeado de forma normal, estamos tomando como referencia un circulo, por lo que solo tenemos que tener en cuenta un radio.
En cambio cuando esto lo hacemos de la forma de un eclipse debemos tener en cuenta 2 radios, la distancia de X y Y, ya que no es el mismo valor que tenemos del centro a arriba que del centro a uno de sus lados
Para dar un redondeado elíptico debemos separar los valores de los radios con un /

Igualmente podemos aplicar los valores a cada un de los lados usando:

  • border-top-left-radius -> Arriba izquierda
  • border-top-right-radius -> Arriba derecha
  • border-bottom-left-radius -> Abajo izquierda
  • border-bottom-right-radius -> Abajo derecha
border-radius: 2rem / 4rem;
border-radius: 2rem 4rem / 4rem 8rem;
border-radius: 3rem 2rem 1rem / 1rem 2rem 3rem;
border-radius: 4rem 3rem 2rem 1rem / 1rem 2rem 3rem 4rem;

Cuando tenemos un cuadrado perfecto podemos hacer que este tome la forma de un circulo completamente

Con el 50% ya tenemos el circulo, no es necesario el 100%. Aunque también es posible

Si quisiéramos un elipse podemos no meternos con tantos estilos y mas bien hacer de nuestro elemento un rectángulo y aplicar el borde como si quisiéramos un circulo

Outline

Es como un borde exterior. Funciona igual que el border aunque solo tenemos el Shorthand, es decir, no tenemos la posibilidad de hacerlo independientemente en cada lado

Esta propiedad no hace parte del modelo de caja, por lo tanto no ocupara ancho

Hacemos referencia a esta linea como si fuese el aura, como una protección intangible

con outline-offset Podemos definir la distancia en que queremos que nuestra outline se aplique
También podemos definir el valor en negativo lo que hara que nuestro outline se vaya hacia adentro de nuestro elemento
outline: thick solid gold; outline-offset: 3rem;
outline-offset: -3rem

Esta propiedad se usa mucho en los inputs, cuando le ponemos el foco
