Funciones en CSS

  • var() -> Lo usamos para llamas nuestras custom properties
  • url() -> Usado para incluir archivos
  • calc() -> Podemos hacer operaciones matemáticas con medidas en CSS
  • min() % max() -> Elige el menor o mayor valor
  • clamp() -> Pasamos 3 valores, el mínimo, el ideal y el máximo. Dentro de esta función también se pueden hacer cálculos

calc

Son importantes los espacios, si no hay espacios entre los operadores y números, no funcionara nuestra función calc 

      
        --padding-size: 2ch;
        --num-ch: 10ch;
        width: calc(var(--num-ch) + var(--padding-size) * 2);

Min Max

        
          width: (300px, 20vw, 20rem);
          height: max(200px, 25vh);

clamp

Nuestro segundo valor al ser dinámico, nuestro texto sera fluido, hasta que llegue a los valores definidos, que en este caso son estáticos

font-size: clamp(2rem, 1rem + 3vw, 3rem);
