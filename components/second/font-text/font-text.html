Font-family

Son un conjunto de estilos de letras que le podemos agregar a nuestro sitio, por defecto es Times Now Roman. Cuando ponemos varias fuentes separadas tomara la que este preparada, esto sera de izquierda a derecha 

      
        

          font-family: Georgia, "Times New Roman";
        

        

          font-family: sans-serif;

Font-size

Tamaño de fuente

      
        

          font-size: 2rem;

Font-weight

Definiremos el grosor, el peso de la tipografía, lo normal es 400 aproximadamente. Entre mas ancha mas se vera en negrilla

Hay algunas fuentes que traen este grosor elevado, por lo que lo mejor seria que si lo queremos poner en negrilla, usemos bold asi este valor elevado por defecto nuestro navegador lo pondrá como en la etiqueta b 

      
        

          font-weight: 400;
        

        

          font-weight: bold;

Font

Shorthand para todas las propiedades, no es muy usado ya que trae muchas y rara vez usamos todas

Debemos poner / entre el font-size y el line-height

El orden de definición es: font-style font-variant font-weight font-size / line-height font-family

Fuentes

Hay diferentes fuentes que están en todos los dispositivos, desde donde sea que nos visite el usuario, están fuentes las tendremos y podemos hacer uso. A estas las conocemos como tipografías seguras

Nuestras fuentes tienen distintos formatos, los mas importantes para nosotros son: woff o woff2 (Web Open Font Format) este es el formato que haremos uso en nuestros documentos, y nuestros sitio web.

El funcionamiento de estas dos propiedades se puede simplificar de la siguiente forma: Digamos que el @font-face es una función con el nombre que le asignamos en el font-family y que esta función la mandamos a llamar en nuestro elemento. Este nombre debe ser único por esto mismo

Font-family

      
        @font-face {
          font-family: "Chalet";
        }

Este nombre lo asignamos nosotros, debe ser segun la fuente y una palabra si es posible, sino separada con -

src

      
        @font-face {
          font-family: "Chalet";
          src: url("Chalet.woff") format ("woff");
        }

Luego debemos poner la ruta de nuestro archivo woff, podemos poner mas rutas por si hay error en esta separando con ",". También es necesario poner el format, ya que asi es como nuestro navegador va a tomar el archivo

Font-display

Esta propiedad nos ayudara el comportamiento de nuestro texto mientras carga la fuente

  • Auto -> Permite que el navegador utilice el método predeterminado, suele ser block
  • Block -> Oculta el texto hasta que la fuente haya sido descargada por completo
  • Swap -> Indica al navegador que utilice la fuente alternativa para mostrar el texto hasta que la fuente personalizada se haya cargado por completo. Si no especificamos ninguna, cargara la de por defecto que en su mayoría es "Times New Roman"
  • Fallback -> Es una mezcla de auto y swap
  • Optional -> ocultara el texto, luego lo cargara con la fuente alternativa y finalmente aplicara la fuente personalizada. La diferencia con el anterior es que este se demora un poco mas con el ciclo
    • El recomendado de usar es Swap

Ejemplo

Definimos nuestro @font-face

        
          @font-face {
            font-family: "Chalet";
            src: url("../../assets/Chalet.woff") format("woff");
            font-display: swap;
          }
De esta forma nuestra familia de fuente ya estaría disponible para nuestro documento

Como es un archivo externo, en algunas ocasiones puede que presente retrasos, dependiendo de la conexión a internet. Lo ideal es que por lo menos pongamos una familia de fuente segura 

        
          font-family: "Chalet", sans-serif;

Fuente externa

Hay una gran variedad de fuentes por lo que hay sitios que ya han organizado estas por nosotros. Cada sitio tiene su forma para que podamos acceder a estas, sin embargo, Google fonts es de las mas grandes ademas de ser open source.

Podemos acceder a sus fuentes totalmente gratis, ademas que podemos hacer uso de los iconos y demás

Ejemplo de uso

Para Google fonts, solo debemos seleccionar la fuente que queremos, este nos dará links que debemos poner en nuestro head que se cargara al entrar a nuestro documento

Raleway

      
        <link rel="preconnect" href="https://fonts.googleapis.com" />
        <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
        <link href="https://fonts.googleapis.com/css2?family=Raleway:wght@400;700&display=swap" rel="stylesheet"/>

Asi ya podríamos usar la fuente en nuestro documento, en este caso solo importamos con el grueso estandar y cuando la queremos en negrilla, también podríamos importarla con todas sus Características per esto pesaría mas y consumiría mas recursos. Por lo tanto es importante elegir variantes que si usemos 

        
          font-family: "Raleway", sans-serif;

Estilos de texto

Text align

Alinear texto
        
          text-align: center;

Text decoration

Controlamos el subrayado
        
          

            text-decoration: underline;

Text ident

Nos permite dar sangria a la primera linea de nuestro texto
        
          text-indent: 1;

text transform

Nos permite transformar nuestro texto
        
          

            Capitalize -> La primera letra de cada palabra estará en mayúsculas
          

          

            uppercase -> Todas las palabras estarán en mayúsculas
          

          

            lowercase -> Todas las palabras estarán en minúsculas
          

          

            none -> No haremos nada con el texto, Por defecto

letter spacing

Definimos cuanto espacio habra ente cada letra
        
          letter-spacing: 0.5rem;

white-space

Definimos que hacer con los espacios en blanco
        
          

            nowrap -> No importa si hay o no, nuestro contenido se vera en una sola linea
          

          

            pre -> El navegador respetara todos los espacios en blanco que le hayamos dado en nuestro documento
          

          

            normal -> El primer espacio sera tomado, ademas, sera wrap, respetara el tamaño de nuestro elemento
          

          

            pre-wrap -> Se adaptara al espacio que le hemos definido a nuestro elemento

Word break

Nos ayudara a definir la forma en que se comportaran las palabras al momento que tengan la necesidad de romperse para pasar a la siguiente linea 
        
          

            normal -> Las palabras que no tengan espacios no se romperán
          

          

            break-all -> Se rompe la palabra, asi quede solo una letra

Writing mode

Controlamos el modo de lectura
        
          

            horizontal-tb -> Por defecto en nuestro idioma
          

          

            vertical-lr ->

Para que esta propiedad funcione es necesario usar overflow y white-space

        
            overflow: hidden;
            white-space: nowrap;
