Tablas en CSS

El atributo border ya esta depreciado, es mejor hacerlo con CSS

border-collapse

Controlamos el espacio que habrá entre nuestras celdas

separate -> hara que entre nuestras celdas hayan 2px de espacio Por defecto

collapse -> Asi no tendrán nada de espacio

Empty-cells

Definimos como queremos que manejen las celdas que están vacías

show -> Aplicara los estilos sin importar que la celda este vacía Por defecto

hide -> ocultara nuestra celda y con ella todos los estilos que le hayamos definido, pero el espacio queda ahi
