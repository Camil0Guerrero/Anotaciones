Imágenes CSS

Imágenes responsivas

Las imágenes son elementos que trabajan en linea

Cuando solo necesitamos que nuestra imagen se adapte a nuestro elemento o que no sobresalga. Con poner un max-width al 100% es suficiente

Imágenes con tamaño específicos

Lo complicado puede que llegue cuando queramos tarjetas o pequeñas muestras, cuando queremos varias imágenes con el mismo tamaño, lo complicado es que no todas tienen las mismas dimensiones. Por ello contamos con propiedades que nos ayudaran a manejar mejor las imágenes

Object fit

Su funcionalidad es parecida a la de background position, nos ayudara a controlar el comportamiento de nuestra imagen dentro del elemento

fill -> Rellenar, de esta forma tratara de cumplir nuestras ordenes, aunque la imagen se deforme

Fill

cover -> No deformara la imagen ya que la hara mas pequeña o grande con sus dimensiones iniciales, sin embargo nuestra imagen no se vera completa, ya que no quedara exactamente

Cover

contain -> Hara que toda nuestra imagen se vea dentro del contenedor, aunque esto implique dejar agujeros

Contain

none -> La imagen queda con las mismas dimensiones, si esta es muy grande, las medidas de nuestro elemento es lo que alcanzaremos a ver. Por defecto

None

scale-down -> Toma el valor que sea menor de los valores contain y none. Este es menos usado ya puede variar el comportamiento segun el viewport

Scale-down

Object position

Definimos la position inicial de nuestra imagen, su manejo y funcionalidad es muy parecidas al de background-position

Para el caso de los ejemplos usare cover para que notemos la diferencia entre las posiciones
